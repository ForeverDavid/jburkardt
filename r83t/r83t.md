R83T\
Tridiagaonl Matrix Utilities {#r83t-tridiagaonl-matrix-utilities align="center"}
============================

------------------------------------------------------------------------

**R83T** is a C++ library which carries out some basic linear algebra
operations on R83T matrices (real 64 bit precision tridiagonal Mx3
format).

### List of Routines: {#list-of-routines align="center"}

-   **I4\_LOG\_10** returns the integer part of the logarithm base 10 of
    an I4.
-   **I4\_MAX** returns the maximum of two I4's.
-   **I4\_MIN** returns the smaller of two I4's.
-   **I4\_POWER** returns the value of I\^J.
-   **R8\_UNIFORM\_01** returns a unit pseudorandom R8.
-   **R83T\_CG** uses the conjugate gradient method on an R83T system.
-   **R83T\_DIF2** returns the DIF2 matrix in R83T format.
-   **R83T\_GS\_SL** solves an R83T system using Gauss-Seidel iteration.
-   **R83T\_INDICATOR** sets the indicator matrix in R83T format.
-   **R83T\_JAC\_SL** solves an R83T system using Jacobi iteration.
-   **R83T\_MTV** multiplies an R83T matrix transposed times an R8VEC.
-   **R83T\_MV** multiplies an R83T matrix times a vector.
-   **R83T\_PRINT** prints an R83T matrix.
-   **R83T\_PRINT\_SOME** prints some of an R83T matrix.
-   **R83T\_RANDOM** returns a random R83T matrix.
-   **R83T\_RES** computes the residual R = B-A\*X for R83T matrices.
-   **R83T\_TO\_R8GE** copies an R83T matrix to an R8GE matrix.
-   **R83T\_ZEROS** zeros an R83T matrix.
-   **R8GE\_MTV** multiplies a vector times an R8GE matrix.
-   **R8GE\_MV** multiplies an R8GE matrix times a vector.
-   **R8GE\_PRINT** prints an R8GE matrix.
-   **R8GE\_PRINT\_SOME** prints some of an R8GE matrix.
-   **R8VEC\_DOT\_PRODUCT** computes the dot product of a pair of
    R8VEC's.
-   **R8VEC\_INDICATOR1\_NEW** sets an R8VEC to the indicator1 vector
    {1,2,3...}.
-   **R8VEC\_NORM** returns the L2 norm of an R8VEC.
-   **R8VEC\_NORM\_AFFINE** returns the affine L2 norm of an R8VEC.
-   **R8VEC\_PRINT** prints an R8VEC.
-   **R8VEC\_UNIFORM\_01\_NEW** returns a unit pseudorandom R8VEC.
-   **R8VEC2\_PRINT** prints an R8VEC2.
-   **TIMESTAMP** prints the current YMDHMS date as a time stamp.
